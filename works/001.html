关于 / About this project

Annieif 是一个个人作品集与数字花园, 既是创作的出口, 也是独立开发者的实验场。整个站点由单个 HTML 文件驱动, 辅以笔记系统与一套可插拔的作品页。它不依赖 CMS, 不使用数据库 — 内容直接由前端渲染, 样式由手工打磨, 数据由访问者自然增长。

从技术上, 这是一个 GitHub Pages 静态站点, 用原生 JavaScript 渲染全部内容, 用 CSS 表达气质与氛围。从表达上, 这是一份自画像 — 有星空、有墨水、有空旷的排版、有留白, 也有温度。

设计 / Design principles

克制。 排版优先, 装饰让步。主色只有三组: 夜的底色、墨水的灰、以及粉色的「温度」。字体系统由 Fraunces 与 Cormorant Garamond 组成 — 一个有锋芒, 一个有呼吸。

留白即语言。 页面之间不堆叠卡片, 而是让文字自己成为结构。章节用序号与时间线锚点共同标记, 形成「左侧目录 + 正文流」的阅读节奏。

动画是叙事的一部分。 大数字有计数动画, 标题有逐字显现, 笔记有展开折叠, 侧边目录随滚动点亮。一切动画都遵循同一条缓动曲线 — 让页面像一次呼吸。

笔记系统 / Notes

这是一个单一数据源的笔记系统。新增一条笔记, 只需在 notes/data.js 写入标题、正文与日期, 其它工作由脚本自动完成: 生成序号、提取首句作为预览、渲染搜索与排序、写入独立的详情页。

阅读者可以按时间排序、按关键词过滤、按章节展开或折叠。目录自动根据内容生成, 章节之间有平滑滚动、锚点与 scroll-spy 高亮。

数据 / The quiet counter

站点用本地计数器追踪访问量与访客数 — 稳定、离线可用、无需第三方。每次加载, 数字都会通过计数动画从 0 缓慢滚到当前值, 像是时间的回声。

下一步 / Next

未来的方向: 更多作品页的写作, 让 Selected Works 从一条清单长成一条长卷; 扩展笔记的标签系统, 让笔记可以按主题聚集成花园的小簇; 以及 — 也许最重要的 — 让站点本身慢慢「变老」。
